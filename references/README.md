# Reference snapshots (local only)

Upstream Claw projects are **not** committed to this repository. Clone them locally for design reference:

```bash
mkdir -p references && cd references

git clone https://github.com/openclaw/openclaw.git openclaw
git clone https://github.com/NousResearch/hermes-agent.git hermes-agent
git clone https://github.com/nanocoai/nanoclaw.git nanoClaw
git clone https://github.com/nullclaw/nullclaw.git nullclaw
git clone https://github.com/sipeed/picoclaw.git picoClaw
```

See [docs/ARCHITECTURE.md](../docs/ARCHITECTURE.md) for how each reference informs BobaClaw.
